# AutoGPT 官方文档汉化

> **来源**: [AutoGPT GitHub](https://github.com/Significant-Gravitas/Auto-GPT)
> **版本**: v0.5.0

## 简介

AutoGPT 是一个实验性的开源应用程序，展示了 GPT-4 语言模型的能力。该程序由 GPT-4 驱动，可以自主地连接 LLM 的思想，以实现你设定的任何目标。

## 快速开始

### 前置要求
- Python 3.10 或更高版本
- OpenAI API Key

### 安装步骤

1. 克隆仓库：
   ```bash
   git clone https://github.com/Significant-Gravitas/Auto-GPT.git
   ```

2. 进入目录：
   ```bash
   cd Auto-GPT
   ```

3. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```
